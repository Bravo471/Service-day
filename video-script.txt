Service Day Dashboard presentation script

1. Introduction
   - The project is a front-end Angular application for a corporate Service Day platform.

2. Employee workflow
   - Employees can log in, browse approved NGO activities, register, change or cancel bookings, and check in using QR codes.

3. Administrator workflow
   - Administrators can create or edit NGO activities, monitor participation, send reminders, and generate QR codes.

4. Technical highlights
   - The application uses Angular routing, feature modules, shared components, services, and mock API data.

5. Conclusion
   - The dashboard demonstrates a complete client-side solution for the assignment requirements.
