# Service Day Dashboard

This Angular application is a front-end prototype for a Service Day Dashboard.

## Features
- Employee login and role-based navigation
- View and register for NGO service activities
- Manage registrations and change/cancel bookings
- Employee QR code check-in
- Admin management for NGO activities
- Participation monitoring and reminder tools
- QR code generation for check-in

## Run Locally
```powershell
cd C:\Users\Bishal Thakuri\Desktop\ServiceDay\service-day-dashboard
npm install
npx ng serve --host 0.0.0.0
```

Open http://localhost:4200/ in your browser.
